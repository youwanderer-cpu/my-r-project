# 🏢 租赁资产全生命周期财务管理系统 (V5.2 审计增强版)

本系统是一款基于 Python 开发的专业财务核算工具，专为 **IFRS 16** 与 **CAS 21（新租赁准则）** 下的租赁会计核算而设计。系统通过 **Streamlit** 提供直观的交互界面，全面覆盖了租赁资产的初始计量、后续摊销、租赁变更重估以及完整的内控审计轨迹追踪。

---

## 🌟 核心财务与审计核算逻辑

系统内置的财务计算引擎严格按照国际会计准则第16号（IFRS 16）以及中国企业会计准则第21号（CAS 21）的合规性要求进行深度优化：

### 1. 精确折现与资金时间价值
* **月度折现率计算**：系统摒弃了传统的“年利率 / 12”的简化名义利率算法，全面采用更符合内控与审计合规要求的**复利有效利率公式**：
    $$r_m = (1 + r_a)^{\\frac{1}{12}} - 1$$
*（其中 r<sub>a</sub> 为年化折现率/增量借款利率，r<sub>m</sub> 为月度折现率）*
* **多频率自动平摊**：支持月付、季付、半年付、年付等多种合同付款频率，系统会自动在底层转换为逐月摊销表。

### 2. 支付时点切换 (Advance vs Arrears)
根据合同约定的支付时点，系统自动切换底层现值（PV）折现逻辑及利息计提基数：
* **期初支付 (Advance)**：现金流发生在周期首月。首笔租金不进行折现，在计算首月利息时，计提基数需先扣除当月已支付额。
* **期末支付 (Arrears)**：现金流发生在周期末月。首笔租金正常折现一次，利息计提基数基于期初完整的负债余额。

### 3. 租赁变更重估 (Lease Modifications)
当合同发生重大条款变更时，系统提供完整的追溯重估机制：
* 支持**扩大租赁范围**、**缩小租赁范围**、**延长或缩短租赁期限**以及**租金对价变更**。
* 系统会根据变更生效日的新折现率与新现金流，自动重估租赁负债现值，并计算出调整差额（$\Delta$ 租赁负债）。
* 同步自动化调增或调减“使用权资产”账面价值，若缩小范围导致资产冲减至零以下，自动将溢出部分计入当期损益（资产处置损益）。

### 4. 税务与核算科目拆解
* **科目明细追踪**：实时计算并展示“租赁付款额总额”与“未确认融资费用”，便于直接对接企业总账中“租赁负债——租赁付款额”与“租赁负债——未确认融资费用”的二级科目分录编制。
* **递延所得税 (Tax & VAT)**：系统自动跟踪资产账面价值（使用权资产净值）与计税基础（租赁负债现值）之间的暂时性差异，动态计算递延所得税资产（DTA）与递延所得税负债（DTL）的变动净额。同时，精准隔离进项税额（VAT）对可折现现金流的影响。

## 🛠️ 技术栈与系统架构

* **前端交互**：Streamlit 响应式 Web 框架
* **数据核算引擎**：Pandas, NumPy, Python-dateutil
* **报表导出引擎**：XlsxWriter (构建多页签高级 Excel 审计底稿)
* **数据持久化**：本地 JSON 数据库 (`leases_db.json` / `users.json`)
* **安全防御**：密码采用 **SHA-256 加盐（Salt）哈希加密** 存储，拒绝明文密码

## 📂 数据库与文件结构说明

整个应用的文件结构应当保持如下布局：

```text
my-r-project/
│
└── lease/
    ├── lease_calculator_VAT_offline_modified_access_audit_final.py  # 核心Streamlit系统
    ├── import_hashlib.py                                            # 权限初始化与用户管理脚本
    ├── leases_db.json                                               # 租赁合同核心台账库
    ├── users.json                                                   # 加盐哈希加密后的用户凭证库
    └── operation_logs.csv                                           # 实时审计追踪日志(全操作留痕)

1. 用户凭证加密库 (users.json)

        "role": "manager",
        "name": "财务经理"
    },
    "user": {

安全提示：初始化脚本内置安全盐值 SALT = "Lease_Project_2026"。默认初始密码分别为 Manager2026 和 Staff2026。

2. 审计留痕日志 (operation_logs.csv)
任何登录、资产新增、租赁变更重估或删除操作，系统都将在底层自动向 operation_logs.csv 追加一条记录，包含：时间、操作用户、用户角色、操作类型 以及 详细改动参数。这为外部审计（如年报审计）提供了强有力的控制测试轨迹。

📊 自动化审计底稿 (Excel 导出结构)
点击系统中的“导出高级 Excel 报表”按钮，系统会调用后端引擎自动生成一个格式高度美化、符合四大审计底稿规范的 .xlsx 工作簿，内含 4 个核心页签（Tabs）：

📊 全周期明细 (Amortization Schedule)：逐月列示自合同开始日至结束日的现金流、期初负债、本期应计利息、本期应付租金、期末负债、使用权资产期初、本期折旧、使用权资产期末净值等 10 余项关键指标。

📋 月末财务快照汇总 (Monthly Summary)：按自然月度进行横向加总，提供全公司视角下的“月末租赁负债余额”、“月末使用权资产净值”以及“当月利息/折旧总额”，用于总账会计直接编制月度结账分录。

🔑 初始合同清单 (Initial Recognition)：抽取所有资产的初始账面信息，包含合同名、现值、未确认融资费用初始总额、初始使用权资产原值。

🔄 租赁变更记录 (Modification Trace)：将所有历史发生的重估记录进行追溯列示，清晰对比变更前后的租金、折现率、到期日以及重估引发的资产负债调整额。

🚀 部署与运行指南
1. 环境依赖安装
在您的本地电脑（或服务器终端）中，运行以下命令安装系统所需的 Python 库：

Bash
pip install streamlit pandas numpy python-dateutil xlsxwriter
2. 首次运行
在终端切换到项目目录并运行主程序：

Bash
cd lease
streamlit run lease_calculator_VAT_offline_modified_access_audit_final.py
3. 生产安全建议
请在生产环境部署前，修改 import_hashlib.py 与主脚本中的 SALT 字符串。
